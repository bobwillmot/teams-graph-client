# Architecture Decision Records

Long-term design decisions and tradeoffs for this project live in this folder.

Conventions:

- Use numbered ADR files such as `0001-topic.md`.
- Keep the standard sections: `Status`, `Context`, `Decision`, and `Consequences`.
- Add a new ADR when a decision changes the project's long-term shape or operating model.
