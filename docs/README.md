# Documentation

Project documentation lives in this folder.

Conventions:

- Use lowercase hyphenated names for Markdown files by default.
- Keep conventional meta-file names such as `README.md` uppercase.
- Store dated agent request and implementation summaries in `agent-transcripts/` using `yyyy-mm-dd.md`.
- Store architecture decision records in `adr/` using numbered kebab-case names such as `0001-short-title.md`.

Folders:

- [agent-transcripts](./agent-transcripts/README.md): dated request and implementation summaries.
- [adr](./adr/README.md): long-term design decisions and tradeoffs.
